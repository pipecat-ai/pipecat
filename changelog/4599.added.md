- Added a `user_audio_preroll_secs` parameter to `OpenAIRealtimeLLMService` controlling how much audio is replayed after an interruption clears the input buffer in locally-driven-turns mode. Defaults to None, auto-sizing the pre-roll from the upstream VAD's `start_secs` (which assumes VAD drives turn starts); set it explicitly when using a non-VAD turn-start strategy.
