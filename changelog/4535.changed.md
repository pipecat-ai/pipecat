- Updated Smallest AI TTS plugin for Waves v4.0.0 API:
  - New WebSocket endpoint `/waves/v1/tts/live` (previously `/waves/v1/{model}/get_speech/stream`)
  - Model is now sent in each message payload instead of the URL, eliminating reconnection on model change
  - Updated model names: `lightning_v3.1` and `lightning_v3.1_pro` (underscore convention)
  - Added `output_format` setting supporting `pcm`, `mp3`, `wav`, `ulaw`, `alaw`
  - Default model changed to `lightning_v3.1_pro` (with `meher` as its default voice)
  - **Breaking**: `SmallestTTSModel.LIGHTNING_V2` removed; `consistency`, `similarity`, `enhancement` settings removed
