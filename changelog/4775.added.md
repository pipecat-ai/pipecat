- `DailyTransport` can now publish a `screenAudio` output track, mirroring `screenVideo`. Add `"screenAudio"` to `DailyParams.audio_out_destinations` (and optionally configure it via `custom_audio_track_params["screenAudio"]`), then write audio frames with `transport_destination="screenAudio"`. Requires `daily-python>=0.29.0`.
