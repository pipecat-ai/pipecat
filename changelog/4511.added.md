- Added support for the Rime `coda` TTS model to `RimeTTSService`, `RimeHttpTTSService`, and `RimeNonJsonTTSService`. The `temperature`, `top_p`, and `repetition_penalty` settings are not used by `coda`. Also added a `timeScaleFactor` setting (for the `arcana` and `coda` models) across all three services — values above 1.0 slow down audio playback; values below 1.0 speed it up.
