- Fixed `UserIdleController` entering a permanent dead state when a user turn ends without producing text (e.g., background noise triggers VAD but STT returns no transcription). The idle timer now also starts on `UserStoppedSpeakingFrame` so that the `on_user_turn_idle` event still fires when no bot response follows. In the normal case, `BotStartedSpeakingFrame` cancels the timer within milliseconds.
