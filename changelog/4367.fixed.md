- Fixed `LLMAssistantAggregator` silently dropping the deferred context frame push when a function-call result arrives during a user-speaking window. The bot-speaking branch already flagged the push and re-triggered it on `BotStoppedSpeakingFrame`; the user-side now mirrors that pattern via a new `_push_context_on_user_stopped_speaking` flag. Previously, a short trigger utterance whose transcription-driven turn-start raced the function-call result could leave the tool response permanently un-delivered — most visible on Gemini Live where the model would sit silent waiting.
