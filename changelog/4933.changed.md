- ⚠️ The default model for `OpenAIResponsesLLMService` and `OpenAIResponsesHttpLLMService` changed from `gpt-4.1` to `gpt-5.4`. Pass `settings=OpenAIResponsesLLMService.Settings(model="gpt-4.1")` (or the HTTP equivalent) to keep the previous model.

  This moves the Responses services off the aging `gpt-4.1` to a current-gen model where reasoning can be enabled without also switching models (this release adds a `reasoning` setting). We default to the full `gpt-5.4` rather than `gpt-5.4-mini`: the mini variant is cheaper and about as fast, but behaves noticeably worse on correctness in our benchmarks. `gpt-5.4` is significantly cheaper than the frontier `gpt-5.5` ($2.50/$15 vs $5/$30 per 1M input/output tokens); it does cost more than `gpt-4.1` ($2/$8), but reasoning is disabled by default (`effort="none"`), so out of the box it behaves like a fast, non-reasoning model — much like the previous `gpt-4.1` default — with reasoning available on demand.
