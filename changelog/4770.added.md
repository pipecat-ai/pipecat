- Added `KeenableWebSearch` (`pipecat.services.keenable.search`), an optional service that gives voice agents live web search. It exposes `search_web` (with optional `site` and date-range filters) and `fetch_page` function tools — pass `search.tools()` to your `LLMContext` and the LLM auto-registers their handlers (no `register_function` call, no async setup). Search runs against a hosted MCP server powered by [Keenable AI](https://keenable.ai), connected lazily on the first call. Install with the `keenable` extra. Works keyless (`pro` mode); set `KEENABLE_API_KEY` for higher rate limits and the lower-latency `realtime` mode (requires an enabled account). Override the mode with `KEENABLE_SEARCH_MODE`.
