- Added `KeenableWebSearch` (`pipecat.services.keenable.search`), an optional service that adds low-latency web search to voice agents via a hosted MCP server powered by [Keenable AI](https://keenable.ai). It wraps `MCPClient`, works keyless by default, and registers search tools on any LLM service. Install with the `keenable` extra. Set `KEENABLE_API_KEY` for higher rate limits and `KEENABLE_SEARCH_MODE=realtime` for the lowest latency.
