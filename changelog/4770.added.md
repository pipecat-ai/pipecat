- Added `KeenableWebSearch` (`pipecat.services.keenable.search`), an optional service that adds web search to voice agents via a hosted MCP server powered by [Keenable AI](https://keenable.ai). It wraps `MCPClient`, works keyless, and registers search tools on any LLM service. Install with the `keenable` extra. Keyless requests use the `pro` mode; set `KEENABLE_API_KEY` for higher rate limits and the lower-latency `realtime` mode (requires an enabled account). Override the mode with `KEENABLE_SEARCH_MODE`.
