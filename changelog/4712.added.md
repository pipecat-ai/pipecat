- Added context carryover support to `AssemblyAISTTService` for Universal-3 Pro streaming (`u3-rt-pro`). A new `agent_context` setting seeds the agent's most recent reply at connect time, and `AssemblyAISTTService.update_agent_context()` updates it mid-session via an `UpdateConfiguration` message (no reconnect). Giving the model the agent's last reply improves transcription of the user's next turn — short answers, spelled-out entities, and similar-sounding words. A `previous_context_n_turns` setting controls how many prior entries are carried forward (set to `0` to disable carryover entirely). U3 Pro features are recognized for the whole `u3-rt-pro` family, including the `u3-rt-pro-beta-1` variant.

- Added `AssemblyAIContextObserver`, a drop-in observer that automatically feeds each bot reply to `AssemblyAISTTService` as context carryover. Add it to your worker with `observers=[AssemblyAIContextObserver(stt)]`. It exposes an `on_agent_context` event handler that fires with the reply text each time context is sent.
