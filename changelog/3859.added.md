- Added SIP/RTP transport (`SIPServerTransport`, `SIPCallTransport`) for accepting incoming SIP calls (dial-in) directly into Pipecat pipelines. Scoped for LAN use with FreeSWITCH. Supports G.711 PCMU/PCMA codecs, 20ms RTP packetization, RFC 2833 DTMF detection, and automatic 8kHz/16kHz resampling.
