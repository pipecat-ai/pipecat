- Added FreeSWITCH SIP/RTP transport (`FreeSwitchSIPServerTransport`, `FreeSwitchSIPCallTransport`) for accepting incoming SIP calls (dial-in) directly into Pipecat pipelines. Supports G.711 PCMU/PCMA codecs, 20ms RTP packetization, RFC 2833 DTMF detection, and automatic 8kHz/16kHz resampling. **Constraints:** LAN-only (no NAT/STUN/ICE), no SIP REGISTER (dial-in only via FreeSWITCH), UAS role only (does not originate calls).
