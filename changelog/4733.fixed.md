- Fixed `ExternalUserTurnCompletionStopStrategy` (and `LLMTurnCompletionUserTurnStopStrategy`) finalizing the user turn while the user was still speaking. The external completion verdict (e.g. an LLM `✓`) can arrive with latency, so the user may have resumed speaking by the time it lands; the strategy now re-validates against live VAD and keeps the turn open instead of letting the bot talk over the user.
