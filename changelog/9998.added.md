- Added `MCPClient.tools()`, a simpler way to use MCP tools: `LLMContext(tools=await mcp.tools())` is now all you need — the client connects on first use, the tools register with the LLM automatically, and the connection is released automatically when the pipeline ends. Optionally, call `close()` (from anywhere, e.g. an `on_client_disconnected` handler) to release the connection earlier. Also added `tools_arguments`, to pin argument values you don't want the model to control (they're applied on every call and hidden from the model), and `call_tool()`, to call an MCP tool directly without going through the LLM.
