- Added `LLMService.run_structured_inference()`, a structured-output sibling of
  `run_inference()` for one-shot, out-of-band calls that returns a validated
  Pydantic model instead of free text. Implemented for OpenAI chat completions
  (`OpenAILLMService` and OpenAI-compatible subclasses), the OpenAI Responses
  services (`OpenAIResponsesLLMService` / `OpenAIResponsesHttpLLMService`), and
  Google Gemini (`GoogleLLMService`). Services without native structured-output
  support (e.g. Anthropic, AWS Bedrock) raise `NotImplementedError`, mirroring
  `run_inference()`.
