- `SileroVADAnalyzer` and `LocalSmartTurnAnalyzerV3` now share a single ONNX `InferenceSession` process-wide across all instances of the same configuration, instead of loading the model weights (~30 MB each) once per instance. The model weights are immutable and ONNX Runtime's `Run()` is thread-safe for concurrent calls on a CPU session, so all per-instance state (audio buffers, executors, Silero recurrent state) stays per-instance while only the session is shared. At high concurrency this removes duplicated weight memory that could OOM a memory-limited deployment, with no latency or throughput regression. The supporting benchmark suite lives in `scripts/benchmarks/onnx_shared_session/`; the change was validated downstream in production before upstreaming.
