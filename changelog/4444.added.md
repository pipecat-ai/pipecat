- Added `GeminiGenAITTSService` for non-streaming Gemini TTS via the `google-genai` `generate_content` API (`response_modalities=["AUDIO"]`). Supports models such as `gemini-3.1-flash-tts-preview`, configurable voice (e.g. `"Kore"`, `"Charon"`), and uses the same `pipecat-ai[google]` extra as the existing Google services. Closes [#4444](https://github.com/pipecat-ai/pipecat/issues/4444).
