- ⚠️ `LLMContextAggregatorPair`'s `realtime_service_mode` is now auto-configured by default. It defaults to `None`: realtime (speech-to-speech) LLM services announce themselves via service metadata and the aggregator turns the mode on automatically. It previously defaulted to `False` (opt-in) — pass `realtime_service_mode=False` to keep the legacy, pre-`realtime_service_mode` context-write behavior.
