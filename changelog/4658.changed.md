- `websockets` is now a core dependency of `pipecat-ai` instead of the `websockets-base` optional extra. The `websockets-base` extra has been removed; service extras that used to pull it in (Cartesia, Deepgram, ElevenLabs, OpenAI, Google, and others) still work unchanged, and `websockets` is now always installed. If you previously installed `pipecat-ai[websockets-base]` directly, just drop the extra since `pip install pipecat-ai` now includes it.
