- Switched the `aws` extra from `aioboto3` to `aiobotocore`. Pipecat only uses the low-level client API, and `aiobotocore` is the async library that `aioboto3` wraps, so depending on it directly drops an unnecessary wrapper layer. AWS service initialization now uses `aiobotocore.session.get_session()` and `session.create_client(...)`; public APIs and credential resolution are unchanged.
