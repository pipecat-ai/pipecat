- Added `RinggSTTService`, a streaming Speech-to-Text integration backed by the official `ringglabs` SDK. Forwards Pipecat `VADUserStartedSpeakingFrame` / `VADUserStoppedSpeakingFrame` events as VAD cues to the server, which uses them for endpointing in `on_final` mode. Install with `pip install "pipecat-ai[ringg]"`.
