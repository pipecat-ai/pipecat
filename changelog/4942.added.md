- Added `KeenableWebSearch` (`pipecat.services.keenable.search`), an optional service that gives voice agents live web search and page reading via a hosted MCP server powered by [Keenable AI](https://keenable.ai). It exposes the server's `search_web_pages` (with optional site and date-range filters) and `fetch_page_content` tools — pass `await search.tools()` to your `LLMContext` and the tools register automatically (the connection is released automatically when the pipeline ends, too). Install with the `keenable` extra. Works keyless (`pro` mode); pass `api_key=` for higher rate limits and access to the lower-latency `realtime` mode (requires an enabled account), selected with `mode="realtime"`.
