Updated the ai-coustics integration to SDK 0.21 (bumped `aic-sdk` to `~=2.5.0`). `AICQuailVADAnalyzer` now reports the model's continuous raw VAD probability (`VadContext.raw_vad_probability()`) gated by Pipecat's `VADParams` instead of a binary speech flag. Because the previous output was binary (`0.0`/`1.0`), `VADParams.confidence` had no effect on this analyzer before — it now governs the speech threshold, so existing `AICQuailVADAnalyzer` users should review their `VADParams.confidence` after upgrading. The ai-coustics voice examples now use the `quail-vf-2.2-l-16khz` enhancement model.
