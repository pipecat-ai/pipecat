- Added subagents support: a worker-based agent framework where multiple workers cooperate around a shared bus. Workers run alongside the main pipeline on a `PipelineRunner`, exchange typed messages and `@job` requests, and can be local, distributed (Redis, PGMQ), or proxied across a WebSocket link. The new code lives in `pipecat.workers`; runnable patterns — handoff, parallel fan-out, sidecar workers, distributed bus, point-to-point proxy, and UI workers — live in `examples/multi-worker/`.
