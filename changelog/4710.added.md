- Added `pipecat init`, which makes a project agent-ready by writing a Pipecat coding-agent guide (`AGENTS.md` plus a `CLAUDE.md` that imports it) into the project, so an AI coding assistant picks up Pipecat conventions automatically and then scaffolds the app with `pipecat create`. Run `pipecat init` (prompts for a directory), `pipecat init my-bot`, or `pipecat init .`; re-running refreshes `AGENTS.md` while preserving an existing `CLAUDE.md` (pass `--force` to overwrite it).
