- Added `KeenableWebSearch` (`pipecat.services.keenable.search`), an optional service that gives voice agents live web search and page reading via a hosted MCP server powered by [Keenable AI](https://keenable.ai). It exposes `search_web` (with optional `site` and date-range filters) and `fetch_page` function tools — pass `search.tools()` to your `LLMContext` and the LLM auto-registers their handlers (no `register_function` call, no async setup). The connection opens lazily on first use. Install with the `keenable` extra. Works keyless (`pro` mode); pass `api_key=` for higher rate limits and the lower-latency `realtime` mode (requires an enabled account), selected with `mode="realtime"`.
